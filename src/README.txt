1º Set up the um web server simples na pasta da visualização.
   Nota: Com o firefox nem é preciso o web server na maioria dos casos mas
         não recomendamos o uso do firefox porque têm uma implementação
         deficiente de SVG e das animações especialmente em termos de performance.	
		 
2º Browser:
    Usar o chrome ou o edge.
	
3º Abrir a página Index.html no browser.

4º Resolução:
	Usar 1920 X 1080 (aka 1080p or full hd) em FULLSCREEN (ex: F11 no chrome).
	
5º Have Fun!!!
